CSS (Cascading Style Sheets) is used to style and design websites. It controls colors, layout, fonts, spacing, and responsiveness. Without CSS, websites would look plain and unstyled.

1. What is CSS?

CSS is a stylesheet language that describes how HTML elements should appear on screen. It separates design from structure.

2. Types of CSS

Inline CSS (inside HTML tag), Internal CSS (inside <style>), External CSS (separate .css file — best practice).

3. Layout Systems

Modern CSS includes Flexbox and Grid, which make it easy to create responsive and complex layouts.

4. Responsive Design

Media queries allow websites to adapt to different screen sizes, making them mobile friendly.

5. CSS Animations & Effects

CSS supports transitions, hover effects, and animations to create engaging user experiences.

CSS turns simple HTML into visually stunning and responsive websites.
